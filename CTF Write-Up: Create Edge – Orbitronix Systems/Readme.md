# README – Create Edge CTF

## Summary

This repository contains the CTF write-up for the Create Edge – Orbitronix Systems challenge. The main objectives were to identify the advertising budget, target audience, and secret marketing tool for Orbitronix Systems.

**Key Results:**

* Advertising Budget: 225,000 USD
* Target Audience: Young Adults
* Secret Marketing Tool: InsightNexus AI

All actions were performed in a controlled CTF environment.
